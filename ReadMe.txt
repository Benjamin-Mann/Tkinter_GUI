Tkinter GUI Password Management Program

Features:
 - Random Password Generation
 - Quick Search
 - Ability To Manage Passwords

Random Password Generator:
Allows user to simply click button on GUI to create an 11 digit password comprised of letters, numbers,
and special characters. Password is automatically saved to clipboard for ease of insertion onto website during sign up.

Quick Search:
All passwords are saved in a .Json file using the website name as the key for searching. All text is converted to
title case therefore user does not need to worry about case sensitivity. The program will return the requested
website with the email/username as well as the password in a separate message box.

Ability To Manage Passwords:
All entered information is uploaded to attached .Json file. Password and Email/Username can be accessed through
the Search button. Removing a password will have to be done locally within the .Json file.
